串口板
------------
串口板用以双系切换板通信和检测机通信。
其中切换板通信线路与检测机通信线路共用一个T型头。

通信方式
=======
使用RS422进行通信，波特率为38400

调试
=====
0. 1621B调试工具接线头的引脚定义与串口板不同，请注意1621B使用T型头的大头的5根引脚，串口板使用T型头小头的4根引脚外加大头的一根地线。
0. 进入串口驱动工作目录后，执行./send_test.py和./recv_test.py可以接收和发送数据。
